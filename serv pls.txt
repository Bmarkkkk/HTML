2. Lépés: Az egyedi konzol (MMC) elkészítése
1.	Miután újraindult a gép, lépj be. (Most már MINTA\Rendszergazda vagy MINTA\Administrator néven fog beléptetni).
2.	Nyomd meg a Win + R gombot a billentyűzeten, írd be: mmc, majd Enter.
3.	Kattints a File -> Add/Remove Snap-in... menüpontra.
4.	A bal oldali listából kattints a DNS-re, majd a középső Add > gombra. A felugró ablakban csak nyomj egy Finish-t (This computer).
5.	Keresd meg a Performance Monitor-t, kattints az Add > gombra, majd Finish.
6.	Keresd meg a Group Policy Management-et, kattints az Add > gombra.
7.	Kattints az OK gombra az ablak alján.
📸 4. KÉP: Csinálj egy képet a teljes ablakról, ahol a bal oldali sávban látni a három hozzáadott modult.
3. Lépés: Szervezeti egységek (OU) felépítése
1.	A Server Managerben kattints a Tools (Eszközök) -> Active Directory Users and Computers opcióra.
2.	A bal sávban kattints a kis nyílra a minta.helyi mellett, hogy kinyíljon.
3.	Kattints jobb gombbal a minta.helyi feliratra -> New -> Organizational Unit.
4.	Név: igazgatóság, majd OK.
5.	Kattints újra jobb gombbal a minta.helyi-re -> New -> Organizational Unit.
6.	Név: mérnökök, majd OK.
7.	Most kattints jobb gombbal a frissen létrehozott mérnökök mappára -> New -> Organizational Unit.
8.	Név: tervezés, majd OK.
9.	Kattints megint jobb gombbal a mérnökök mappára -> New -> Organizational Unit.
10.	Név: gyártás, majd OK.
📸 5. KÉP: Kattints rá a bal oldali fában a mérnökök mappára, hogy kinyíljon, és látszódjon a fa struktúrája (igazgatóság, mérnökök, azon belül tervezés és gyártás). Fotózd le!
